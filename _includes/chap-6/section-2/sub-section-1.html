As for the "Analysis" mode object(s) can be selected using the mouse left click. In "Edition" mode selected atoms/objects are subjects to motion interactions using the mouse and it becomes complicated to check motion information with visual measurements. However using the "Tools" menu it is possible to activate the "Measures" selection mode (available when using the "Model edition" dialog (see section 6.1.2.7):
